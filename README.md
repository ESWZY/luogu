# 哗啦啦 2.1 重磅推出！

## What's new

- 修改了部分锣鼓音源
- 增加月琴功能
- 增加编辑模式，可预设锣鼓经与琴谱，可以拖拽到乐池中自动播放
- 增加锣鼓月琴的导入导出功能

## Usage:
- 播放：将预设锣鼓拖拽到乐池中，通过弹出框修改参数然后点击乐池中的播放按钮播放
- 导入：点击乐池顶部的“导入”按钮，导入乐谱文件（例如示例中的文昭关），然后播放
- 导出：点击乐池顶部的“导出”按钮，将所有锣鼓琴谱导出为下次能导入的文本到剪贴板中，再粘贴到txt内保存即可
